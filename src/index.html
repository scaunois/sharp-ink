Pour utiliser ce site, vous devez activer Javascript. Pour parler technique, il est construit sur le framework Angular, qui fonctionne lui-même avec Javascript. Donc si vous désactivez Javascript, il ne peut tout simplement pas fonctionner ! :)
Rassurez-vous, ici, Javascript n'est aucunement utilisé pour vous pister, mais simplement pour vous offrir une expérience de navigation moderne, un peu plus agréable qu'un vieux blog sans grande dynamique...
